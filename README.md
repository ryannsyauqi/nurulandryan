# Local Development

To run this project locally:

1. **Install Dependencies** (if you haven't already):
   ```bash
   npm install
   ```

2. **Start Local Server**:
   ```bash
   npm run dev
   ```

The site will be available at `http://localhost:5173`.

> [!NOTE]
> This project uses [Vite](https://vitejs.dev/) for local development. It provides Hot Module Replacement (HMR) and a fast development experience.
